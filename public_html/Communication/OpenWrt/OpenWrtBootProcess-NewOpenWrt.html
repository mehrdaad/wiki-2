Overview

Secure boot

PBL(Primary Boot Loader) SBL1(Second BootLoader stage 1) 初始化buses、DDR、clocks等 QSEE/TrustZone 安全相关 SBL1移交运行控制权给QSEE。QSEE建立安全运行环境，配置xPU，支持fuse。

APPSBL 即为BootLoader HLOS(High LevelOperating System) 即为Linux/
