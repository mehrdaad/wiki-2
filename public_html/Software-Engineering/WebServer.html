Cloud Database and Open Source DBs 云数据库和开源的DB

xeround

xeround是能取代现有的MySQL数据库，并且提供无缝的MySQL的可扩展性和高可用性的服务。xeround运行在云中，允许您扩展数据库或自动退出，即使在发生故障时的也能保持可用性。你可以运行亚马逊Web服务和Rackspace公司的数据库，惠普云服务等。http://xeround.com/

MySQL的衍生版本 - Percona、MariaDB、MySQL branch at Google

MySQL是世界上最流行的开源数据库。无论你是技术的ISV还是大型企业，MySQL 可以经济有效地帮助您提供高性能，进行可伸缩的数据库应用。

  • Percona - Experts in MySQL, InnoDB, and LAMP Performance
  • MariaDB - MySQL 原創辦人繼續製作、維護的產品
  • MySQL branch maintained and used at Google (google-mysql)
  • twitter/mysql - Twitter 的 MySQL branch 套件.

ouchDB是一个数据库，它完全拥抱网络，使用JSON文档存储您的数据存储，网页浏览器访问您的文件，使用JavaScript进行查询，整合和转化。 http://couchdb.apache.org/

Neo4j的是一个高性能的的NOSQL图形数据库，具有一个成熟和强大的数据库的所有功能。 http://neo4j.org/
